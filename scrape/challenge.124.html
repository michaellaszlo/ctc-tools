Aug 2015 - Free Spirits 9th Anniversary
This challenge was chosen by Free Spirits

Row 3 x 9mins with 3 minutes rest between each interval.


Record the total distance for the 27 minutes (not including any resting metres).


There are no stroke rate restrictions.
First interval is a standing start and rests can be active or passive,
but the piece MUST be done as written ie 3 x 9 minutes and NOT a straight 27 minutes.

The August challenge will be chosen by the Gee Crew.


See the ISS forum thread for details.
